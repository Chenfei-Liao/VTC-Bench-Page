# VTC-Bench Project Page

This is the project page for "Are We Using the Right Benchmark? An Evaluation Framework for Visual Token Compression Methods".

## Adding the Paper PDF

To add the paper PDF to this repository:

1. Compile the LaTeX source from `../latex/` directory to generate `paper.pdf`
2. Copy the generated PDF to this directory:
   ```bash
   cp ../latex/paper.pdf .
   ```
3. Commit and push:
   ```bash
   git add paper.pdf
   git commit -m "Add paper PDF"
   git push origin main
   ```

The homepage (`index.html`) is already configured to link to `paper.pdf`.

## Repository Structure

- `index.html` - Main project page
- `paper.pdf` - Paper PDF (to be added)
- `README.md` - This file

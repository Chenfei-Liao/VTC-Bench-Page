An Evaluation Framework for Visual Token Compression Methods

Abstract

Recent endeavors to accelerate inference in Multimodal Large Language Models (MLLMs) have primarily focused on visual token compression. The effectiveness of these methods is typically assessed by measuring the accuracy drop on established benchmarks. However, these benchmarks are originally designed to assess perception and reasoning capabilities, not compression efficiency.

Strikingly, our investigation reveals that simple image downsampling consistently outperforms many advanced compression methods across multiple widely used benchmarks.

Motivated by these findings, we introduce VTC-Bench, an evaluation framework that incorporates a data filtering mechanism to denoise existing benchmarks, thereby enabling fairer and more accurate assessment of visual token compression methods.

Motivation: The Simplicity Bias

"Are current benchmarks difficult enough to evaluate visual token compression?"

Figure 1. The Anomaly. (a) Average Decline Ratio (ADR) of five visual token compression methods on eight benchmarks. (b) Inference time comparison.
Surprisingly, simple downsampling (the baseline) outperforms sophisticated methods on most general benchmarks.

We conducted a comprehensive empirical study and found a counterintuitive phenomenon: Simple image downsampling consistently outperforms advanced compression methods (like FastV, VisionZip) across multiple popular benchmarks (GQA, MMBench, etc.).

Why? Because existing benchmarks contain many "simple" samples that can be answered correctly even with very low-resolution images. These samples act as noise, masking the true performance differences between compression methods.

VTC-Bench Framework

Figure 2. VTC-Bench Overview. We propose using downsampling as a difficulty filter. Samples are categorized into Group A (Difficult) and Group B (Simple) based on whether the downsampling baseline can answer them correctly.
